A way for grocery stores to engage omnichannel shoppers

This is a machine learning project that I built in Python using PyTorch. Using the Fruits360 dataset, I was able to program a model that can classify fruits, vegetables, and other produce items and produce a QR code for customers to scan. This QR code then leads to a curated database of recipes that can be made using the produce item. This engages omnichannel shoppers (people who shop both online and in-store) by providing exciting technology for them to use in-store. To learn more about the code behind the project as well as why it's important, read the article above.
